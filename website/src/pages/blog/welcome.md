# Castla 출시 — Tesla 대화면에서 Android 앱을 즐기세요

<time>2026년 3월 24일</time>

---

Tesla를 타면서 "내 폰 화면을 저 큰 화면에 띄울 수 없을까?"라고 생각해본 적 있으신가요? **Castla**는 바로 그 문제를 해결합니다.

## Castla가 하는 일

Android 폰의 전체 화면을 Tesla 브라우저로 실시간 스트리밍합니다. 별도 PC 연결 없이, 같은 WiFi(또는 폰 핫스팟)에 연결된 상태에서 Tesla 브라우저에 URL을 입력하고 4자리 PIN을 넣으면 바로 미러링이 시작됩니다.

## 핵심 기능 한눈에 보기

### 실시간 H.264 스트리밍
WebSocket 기반으로 H.264 하드웨어 인코딩된 영상을 전송합니다. 최대 **1080p 60fps**(PRO)까지 지원하며, 네트워크 상태에 따라 1~15Mbps 범위에서 **자동 비트레이트 조절**(ABR)이 이루어집니다. H.264를 지원하지 않는 환경에서는 MJPEG로 자동 폴백됩니다.

### 가상 디스플레이 (Shizuku)
Shizuku가 설치되어 있으면 **독립적인 가상 디스플레이**를 생성합니다. 덕분에 Tesla에 화면을 보내는 동안에도 폰은 정상적으로 사용할 수 있습니다. 폰 화면을 꺼도 Tesla 쪽 가상 디스플레이는 계속 유지됩니다.

### Tesla에서 직접 앱 실행 — 앱 런처
미러링이 시작되면 Tesla 브라우저에 **카테고리별 앱 런처**가 표시됩니다. 내비게이션, 동영상, 음악, 일반 앱으로 분류된 목록에서 원하는 앱을 탭하면 가상 디스플레이 위에서 바로 실행됩니다.

### 터치 컨트롤
Tesla 화면에서 직접 터치로 폰을 조작합니다. **멀티터치** 지원(스와이프, 탭, 스크롤)은 물론, 텍스트 입력이 필요할 때는 **플로팅 입력 버블**이 나타나 검색이나 로그인도 Tesla 화면에서 바로 할 수 있습니다. 한국어/일본어 등 CJK 조합 입력도 지원됩니다.

### DRM 앱 웹 브라우저 모드
Netflix, YouTube, Disney+ 같은 **DRM 보호 앱**은 화면 캡처가 차단되어 일반 미러링으로는 검은 화면만 나옵니다. Castla는 이런 앱을 탭하면 iPad User-Agent의 **웹 브라우저**를 자동으로 열어 웹 버전으로 전체화면 재생할 수 있게 합니다. Wavve, Tving, 쿠팡플레이 등 국내 OTT도 지원합니다.

### 오디오 스트리밍
Android 10 이상에서 **디바이스 오디오**도 함께 전송합니다. 영상과 소리가 동기화되어 Tesla 쪽에서 소리까지 들을 수 있습니다.

### Tesla 자동 감지
Bluetooth를 통해 Tesla 차량을 자동으로 감지합니다. Tesla에 연결되면 알림이 뜨고, 탭 한 번으로 미러링을 바로 시작할 수 있습니다.

## PRO에서 추가되는 것

무료 버전은 **720p 30fps**로 사용 가능하며, PRO를 구매하면:
- **1080p 해상도** & **60fps** 해금
- 동영상/음악 카테고리 앱 실행 잠금 해제
- 한 번 결제로 영구 사용

## 시작하기

1. [Google Play에서 Castla 다운로드](https://play.google.com/store/apps/details?id=com.castla.mirror)
2. [Shizuku 설치](/setup) (터치 입력 & 가상 디스플레이에 필요)
3. 폰과 Tesla를 같은 WiFi에 연결
4. Castla에서 "Start Mirroring" → Tesla 브라우저에서 URL 입력 → PIN 입력 → 완료!

자세한 사용법은 [사용 방법 안내](/guide)를 참고하세요.
